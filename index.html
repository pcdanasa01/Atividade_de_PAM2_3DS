3. Manipulação da DOM

Demonstrar e explicar meios de manipulação de elementos da DOM.

3.1. getElementById()

Usado para retornar um objeto na DOM que possua o id especificado. Nesse exemplo, o quadrado verde possui um id, o qual será usado para adicionar uma borda em sua volta.

3.2. getElementsByTagName()

Identifica e retorna uma coleção de elementos da DOM que possuam o nome da tag especificado. Nesse exemplo, essa função será usada para retornar as tags <p> presentes nesse documento e mostrar o seu número.

Total de tags <p>: 0

3.3. getElementsByClassName()

Identifica e retorna uma coleção de elementos da DOM que possuam o nome da classe especificado. Como anteriormente, será utilizada para contar o número de elementos da DOM que possuem a classe inline-block.

Total de elementos com 'inline-block': 0

3.4. querySelector()

Retorna o primeiro elemento que se enquadra com o seletor especificado. Dois dos quadrados abaixo possuem duas classes únicas, o qual apenas o primeiro elemento que a possui uma delas receberá uma borda.

3.5. querySelectorAll()

Retorna todos os elementos que se enquadram no seletor especificado. Como no anterior, dois dos quadrados abaixo possuem duas classe únicas, com isso os quadrados que possuem uma das classes listados no seletor, teram uma borda.

4. Eventos de ação

Demonstrar eventos de ação com elementos da DOM.

4.1. onclick()

Esse evento sofre um gatilho quando o objeto no qual ele está agindo é pressionado. Clique nesse texto para algo acontecer.

4.2. onmouseenter()

Esse evento sofre um gatilho quando o ponteiro do mouse entra na área do objeto em que ele está agindo. Coloque o mouse dentro da área do texto.

4.3. onmouseleave()

Esse evento sofre um gatilho quando o ponteiro do mouse sai do espaço do elemento em que ele está agindo. Coloque o mouse dentro da área do texto, e depois tire.
